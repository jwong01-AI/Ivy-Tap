Google Cloud setup (one-time)
1. console.cloud.google.com → APIs & Services → Enable Google Sheets API
2. Credentials → Create → OAuth 2.0 Client ID → Web application
3. Authorised JavaScript origins: add the URL you open this file from
4. OAuth consent screen → add both your Gmail & Ivy's Gmail as Test users
5. The spreadsheet columns must stay: Date | Credit | Debit | Current Amount | Comments
